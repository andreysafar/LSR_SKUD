######################################
# Пакет для интеграции с ParsecNET 3 #
######################################

Описание пакета:

Файл "Doc\ParsecIntegrationDescription_<date>.pdf" содержит описание всех функций интеграционного
сервиса ParsecNET 3.

Файл "Doc\ParsecIntegrationDescription_<date>.en.pdf" содержит описание всех функций интеграционного
сервиса ParsecNET 3 (документ на английском языке).

Файл "Doc\EventHistory_comments.txt" содержит описание и комментарии к механизму получения событий из системы ParsecNET 3 с помощью интеграционного сервиса.

Файл "Doc\События ParsecNET 3.xlsx" содержит список событий, формируемых системой ParsecNET 3.

Файл "Doc\Руководство по разработке C# скриптов для модуля автоматизации ParsecNET <>" содержит описание требований для разработчиков скриптов для инструмента "Редактор задач" ПО PNSoft.

Examples:
* Папка "ParsecIntegrationClient_C#" содержит исходный код (C#) тестового клиента к сервису
интеграции ParsecNET-3 и откомпилированный пример.
* Папка "DelphiClient" содержит исходные коды примера работы с интеграционным сервисом из среды Borland Delphi XE5.
* Папка "php_call_example" содержит пример корректного вызова функции интеграционного сервиса из php-скрипта.
* Папка "python_exampe" содержит пример корректного вызова функции интеграционного сервиса из скрипта на языке Python.
* Папка "python_event_callback" содержит пример подписок на события различными методами (GET, POST) на языке Python.

Папка "bin" содержит откомпилированные примеры указанных выше демонстрационных приложений

В примерах использованы не все функции сервиса, однако на базе примеров должно 
быть понятно, как осуществляется работа с сервисом.
